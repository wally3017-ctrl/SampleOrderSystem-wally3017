# 반도체 시료 생산주문관리 시스템

## 목적

반도체 회사 S-Semi의 시료(Sample) 생산 및 주문 흐름을 체계적으로 관리하기 위한 **콘솔 기반 애플리케이션**이다.

기존에 엑셀·메모장으로 관리하던 주문·재고·공정 현황을 디지털화하여, 주문 접수부터 생산·출고까지의 전 과정을 하나의 시스템에서 추적할 수 있도록 한다.

### 주요 기능

| 기능 | 설명 |
|------|------|
| 시료 관리 | 시료 등록, 목록 조회, 검색 |
| 시료 주문 | 고객 주문 접수 (RESERVED) |
| 주문 승인/거절 | 재고 상황에 따라 CONFIRMED 또는 PRODUCING 자동 전환 |
| 생산라인 | FIFO 기반 단일 생산라인, 실 생산량 자동 계산 |
| 모니터링 | 상태별 주문 수 및 시료별 재고 현황 |
| 출고 처리 | CONFIRMED 주문을 RELEASED로 전환 |

### 주문 상태 흐름

```
RESERVED → (승인) → 재고 충분: CONFIRMED → RELEASED
                 → 재고 부족: PRODUCING → CONFIRMED → RELEASED
         → (거절) → REJECTED
```
